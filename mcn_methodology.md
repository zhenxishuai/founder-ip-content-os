# MCN 级网感拆解与内容生成方法论

基于《【MCN训练网感】一周自媒体素材拆解.pdf》提取的核心标准：

## 1. 核心认知
- **网感不是天赋，是系统训练的方法论**。
- **不要指望灵感，要指望系统**。
- **爆款有迹可循**：通过系统分析复制爆款能力。

## 2. 四大训练法
1.  **分析选题**：去搜索栏搜这个选题，看同类有没有爆过。如果同类都爆了，说明选题一定好。
2.  **训练开头（3秒钩子）**：花一天写文案，却只花几分钟写开头，是白忙活。开头不吸引人，观众根本看不到后面。
    - **公式**：否定共识 + 给出新知。
    - **示例**：“很多人以为网感是天赋，错了，网感是可以训练的。”
3.  **分析框架**：把爆款视频拆成一句一句，用表格写出来，看清楚结构。
4.  **建立素材库（弹药库）**：平时看到好东西就往里丢（选题库、金句库、案例库、标题库）。打仗时直接从弹药库拿，而不是临时造子弹。

## 3. 内容结构标准
- **短视频脚本结构**：
    - **【开头】**：3秒钩子，打破认知，吸引留存。
    - **【中间】**：干货/方法论拆解（1, 2, 3, 4 点），逻辑清晰。
    - **【结尾】**：引导互动（如“评论区扣1”），强化认知。
- **小红书/公众号结构**：
    - **开头 -> 发展 -> 转折 -> 收尾**。
    - 标题必须包含爆款元素（如“MCN总监揭秘”、“亲测有效”、“XX的N个核心方法”）。

## 4. 爆款标题模板
- 「为什么你写的XX留不住人？」
- 「做XX总不火？你可能犯了这N个错误」
- 「XX揭秘：XX不是天赋，是训练出来的」
- 「XX总结：XX的N个核心方法 | 亲测有效」

## 5. 写作风格要求
- **拒绝 AI 味**：剔除“此外、至关重要、深入探讨”等词汇。
- **具体化**：用事实、数据、细节代替抽象概括。
- **节奏感**：混合长短句，口语化，带情绪色彩。
- **第一人称**：表达真实观点和反应。
