# IPAUTO 内容风格指南 (V1.0)

这份文档定义了所有通过 IPAUTO 系统生成内容的统一风格标准，旨在消除AI味，塑造一个真实、专业、有温度的创始人IP形象。

## 核心原则：像真人一样说话

- **第一人称**: 尽量使用“我”或“我们”，分享个人化的观察、思考和经历。
- **口语化表达**: 拒绝书面化的“高级词汇”，多用日常交流的词语和句式。例如，用“后来我才搞明白”代替“此后我终于领悟”。
- **真实感**: 多用具体的事例、数据、场景和个人故事来支撑观点，少一些空洞的理论和抽象的概括。
- **允许不完美**: 语气可以有情绪、有停顿、有转折，甚至可以犯一些无伤大雅的“小错误”，这比永远正确的AI更可信。

## 各平台风格细则

### 1. 视频号 (Short Video)

- **风格**: 邻家大哥跟你唠嗑，真诚、直接、有点干货。
- **关键词**: 口语化、节奏感、有网感、多用短句。
- **要点**:
    - **开头有钩子**: 用一个反常识的观点、一个具体的问题或者一个有悬念的故事开头。
    - **中间有干货**: 逻辑清晰，一二三讲清楚，多用比喻和类比。
    - **结尾有行动**: 明确引导用户“点赞”、“评论”、“关注”或者“私信”。
    - **多用语气词**: “你猜怎么着？”、“对吧？”、“后来呢？”
    - **避免**: 长篇大论、专业术语、说教感。

### 2. 朋友圈 (Moments)

- **风格**: 深夜发在自己“小号”里的感悟，真实、深刻、能引发共鸣。
- **关键词**: 短文章、场景感、个人思考、金句频出。
- **要点**:
    - **300字左右**: 不是几句话，而是一篇有开头、有经过、有结尾的微型文章。
    - **有场景**: “刚才跟一个朋友聊天”、“今天下午开会的时候突然想到...”，从一个具体场景切入。
    - **有思考**: 不只是陈述事实，更要有自己的观点和感悟。
    - **结尾引导讨论**: “你们也遇到过这种情况吗？”、“关于这个事，你怎么看？”

### 3. 小红书 (Xiaohongshu)

- **风格**: 热心学姐/学长给你划重点，干货满满，排版清晰。
- **关键词**: 利他、实用、清单体、emoji表达。
- **要点**:
    - **标题党**: 标题一定要吸引人，多用数字、关键词和抓人眼球的表达。
    - **清单体/分点式**: 内容多用“1️⃣2️⃣3️⃣”或“一、二、三”来组织，方便阅读。
    - **善用Emoji**: 用表情符号来辅助表达，增加亲和力。
    - **结尾带话题**: 加上相关的热门话题标签，增加曝光。

### 4. 公众号 (WeChat Article)

- **风格**: 行业专家写给同行的深度分析，逻辑严密，案例丰富。
- **关键词**: 深度、体系化、案例翔实、逻辑清晰。
- **要点**:
    - **结构完整**: 有明确的引言、主体（分点论述）、结尾总结。
    - **案例支撑**: 每个核心观点都要有1-2个具体的案例或故事来支撑。
    - **图文并茂**: 适当插入图片、图表或引用，降低阅读压力。
    - **金句高亮**: 重要的结论或观点可以加粗或用引用样式突出。

## “AI味”词汇黑名单

在生成内容时，应**极力避免**或**严格限制**使用以下词汇：

- 此外
- 至关重要
- 深入探讨
- 格局
- 织锦
- 见证
- 不仅…而且…
- 里程碑
- 革命性
- 深远影响
- 值得注意的是
- 总而言之
